pytablewriter-altrow-theme is a `pytablewriter <https://github.com/thombashi/pytablewriter>`__ plugin to provide a theme that colored rows alternatively.


Installation
============================================
::

    pip install pytablewriter-altrow-theme

Usage
============================================

:Sample Code:
    .. code-block:: python

        import pytablewriter as ptw

        writer = ptw.TableWriterFactory.create_from_format_name(
            "markdown",
            headers=["INT", "STR"],
            value_matrix=[
                [1, "hoge"],
                [2, "foo"],
                [3, "bar"],
            ],
            margin=1,
            theme="altrow",
        )
        writer.write_table()

:Sample Code:
    .. code-block:: python

        import pytablewriter as ptw

        writer = ptw.TableWriterFactory.create_from_format_name(
            "markdown",
            headers=["INT", "STR"],
            value_matrix=[
                [1, "hoge"],
                [2, "foo"],
                [3, "bar"],
            ],
            margin=1,
            theme="altrow",
        )

        writer.set_theme("altrow", color="yellow")

        writer.write_table()
